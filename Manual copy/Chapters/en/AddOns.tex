\chapter{Add Ons}

\subsection{Proposed Add-Ons}

\begin{itemize}
	\item \textbf{LoRa or Wi-Fi Module for Remote Access:}  
	Enables wireless communication between multiple Jetson units or a central monitoring dashboard.  
	This can be implemented using the ESP32 module or Seeed Studio Wio-E5 LoRaWAN board for long-range communication.
	
	\item \textbf{Edge AI Model Optimization:}  
	Integrate TensorRT or NVIDIA DeepStream SDK to reduce inference latency and improve FPS during peak workloads.
	
	\item \textbf{Cloud Synchronization:}  
	Store plate detections and images on a secure AWS S3 or Firebase database for remote analytics and redundancy.
	
	\item \textbf{Automated Billing / Access Control:}  
	Extend the database layer to integrate with RFID or barrier-gate control for smart parking applications.
	
	\item \textbf{Thermal Monitoring Sensor:}  
	Attach a DS18B20 or MLX90614 temperature sensor to monitor the Jetson Nano’s internal heat, triggering cooling when necessary.
\end{itemize}
