◆ cudaColormapInit()

cudaError_t cudaColormapInit ( )

Initialize the colormap palettes by allocating them in CUDA memory.

Note
cudaColormapInit() is automatically called the first time that any colormap is used, so it needn't be explicitly called by the user unless they wish to do it at start-up.
Returns
cudaSuccess on success or if already initialized.
