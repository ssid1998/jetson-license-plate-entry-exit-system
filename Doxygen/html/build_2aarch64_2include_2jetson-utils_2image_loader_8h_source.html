33/**
34 * Load an image or set of images from disk into GPU memory.
35 *
36 * Supported image formats for loading are JPG, PNG, TGA, BMP, GIF, PSD, HDR,
37 * PIC, and PNM (PPM/PGM binary). Internally, imageLoader uses the loadImage()
38 * function to load the images, so the supported formats are the same.
39 *
40 * imageLoader has the ability to load an sequence of images from a directory,
41 * including wildcard characters (e.g. `images/*.jpg`), or just a single image.
42 * When given just the path to a directory, it will load all valid images from
43 * that directory.
44 *
45 * @note imageLoader implements the videoSource interface and is intended to
46 * be used through that as opposed to directly. videoSource implements
47 * additional command-line parsing of videoOptions to construct instances.
48 *
49 * @see videoSource
50 * @ingroup image
51 */
