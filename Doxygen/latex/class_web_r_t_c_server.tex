\doxysubsection{Detailed Description}
Web\+RTC signalling server for establishing and negotiating connections with peers for bi-\/directional media streaming. Users can bind routing functions that configure their pipelines dynamically to handle incoming/\+outgoing streams.

It uses websockets to handle the communication with clients for SDP offers, ICE messages, ect. \begin{DoxySeeAlso}{See also}
\href{https://doc-kurento.readthedocs.io/en/stable/_static/client-javadoc/org/kurento/client/WebRtcEndpoint.html}{\texttt{https:\+/\+/\+doc-\/kurento.\+readthedocs.\+io/\+en/\+stable/\+\+\_\+static/\+client-\/javadoc/\+org/\+kurento/\+client/\+\+Web\+Rtc\+Endpoint.\+html}}
\end{DoxySeeAlso}
By default it also serves simple HTML for viewing video streams in browsers, but for full hosting you\textquotesingle{}ll want to run this alongside an actual webserver.
