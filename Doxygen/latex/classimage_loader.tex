\doxysubsection{Detailed Description}
Load an image or set of images from disk into GPU memory.

Supported image formats for loading are JPG, PNG, TGA, BMP, GIF, PSD, HDR, PIC, and PNM (PPM/\+\+PGM binary). Internally, \doxylink{classimage_loader}{image\+Loader} uses the \doxylink{group__image_gab2959819259e3cbcfd72ad633f8f9f17}{load\+Image()} function to load the images, so the supported formats are the same.

\doxylink{classimage_loader}{image\+Loader} has the ability to load an sequence of images from a directory, including wildcard characters (e.\+g. {\ttfamily images/\+\texorpdfstring{$\ast$}{*}.jpg}), or just a single image. When given just the path to a directory, it will load all valid images from that directory.

\begin{DoxyNote}{Note}
\doxylink{classimage_loader}{image\+Loader} implements the \doxylink{classvideo_source}{video\+Source} interface and is intended to be used through that as opposed to directly. \doxylink{classvideo_source}{video\+Source} implements additional command-\/line parsing of \doxylink{structvideo_options}{video\+Options} to construct instances.
\end{DoxyNote}
