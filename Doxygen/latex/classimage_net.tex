void image\+Net::\+\+Set\+Smoothing (\begin{DoxyParamCaption}\item[{float}]{factor}{}\end{DoxyParamCaption})\hspace{0.3cm}{\ttfamily [inline]}}

Enable temporal smoothing of the results using EWMA (exponentially-\/weighted moving average). This filters the confidence values of each class over \texorpdfstring{$\sim$}{\string~}N frames to reduce noise and jitter. In lieu of storing a history of past data, this uses an accumulated approximation of EMA:\+

EMA(x,t) = EMA(x, t-\/1) + w \texorpdfstring{$\ast$}{*} (x -\/ EMA(x, t-\/1))

where x is a class softmax output logit, t is the timestep, and w is the smoothing weight.


\begin{DoxyParams}{Parameters}
{\em factor} & either a weight between \+[0,1]\+ that\textquotesingle{}s placed on the latest confidence values, or the smoothing window as a number of frames (where the weight will be 1/\+N). ~\newline
 For example, a factor of N=5 would average over approximately the last 5 frames, and would be equivalent to specifying a weight of 0.\+2 (either can be used). A weight closer to 1 will be more responsive to changes, but also more noisy. ~\newline
 Setting this to 0 or 1 will disable smoothing and use the unfiltered outputs.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
this can also be set using the {\ttfamily -\/-\/smoothing=N} command-\/line argument.
