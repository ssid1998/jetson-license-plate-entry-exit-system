When the previous data import job is complete, return to the DIGITS home screen. Select the {\ttfamily Models} tab and choose to create a new {\ttfamily Segmentation Model} from the drop-\/down:\+



In the model creation form, select the dataset you previously created. Set {\ttfamily Subtract Mean} to None and the {\ttfamily Base Learning Rate} to {\ttfamily 0.\+0001}. To set the network topology in DIGITS, select the {\ttfamily Custom Network} tab and make sure the {\ttfamily Caffe} sub-\/tab is selected. Copy/\+paste the {\bfseries{\href{https://raw.githubusercontent.com/NVIDIA/DIGITS/master/examples/semantic-segmentation/fcn_alexnet.prototxt}{\texttt{FCN-\/\+Alexnet prototxt}}}} into the text box. Finally, set the {\ttfamily Pretrained Model} to the output that the {\ttfamily net\+\_\+surgery} generated above:\+ {\ttfamily DIGITS/\+examples/\+semantic-\/segmentation/\+fcn\+\_\+alexnet.\+caffemodel}



Give your aerial model a name and click the {\ttfamily Create} button at the bottom of the page to start the training job. After about 5 epochs, the {\ttfamily Accuracy} plot (in orange) should ramp up and the model becomes usable:\+



At this point, we can try testing our new model\textquotesingle{}s inference on some example images in DIGITS.\hypertarget{md_jetson-inference_2docs_2segnet-training_autotoc_md424}{}\doxysubsubsection{\texorpdfstring{Testing Inference Model in DIGITS}{Testing Inference Model in DIGITS}}\label{md_jetson-inference_2docs_2segnet-training_autotoc_md424}
Before transfering the trained model to Jetson, let\textquotesingle{}s test it first in DIGITS. On the same page as previous plot, scroll down under the {\ttfamily Trained Models} section. Set the {\ttfamily Visualization Model} to {\itshape Image Segmentation} and under {\ttfamily Test a Single Image}, select an image to try (for example {\ttfamily /\+\+NVIDIA-\/\+Aerial-\/\+Drone-\/\+Dataset/\+\+FPV/\+\+SFWA/\+720p/\+images/\+0428.png}):\+



Press {\ttfamily Test One} and you should see a display similar to:\+
