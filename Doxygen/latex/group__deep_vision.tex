\doxysubsection{Detailed Description}
C++ runtime library for supporting deep vision neural network models and inferencing with Tensor\+RT. Available network primitives currently include:\+ ~\newline

\begin{DoxyItemize}
\item \doxylink{group__image_net}{image\+Net} for image recognition
\item \doxylink{group__detect_net}{detect\+Net} for object detection + localization
\item \doxylink{group__seg_net}{seg\+Net} for semantic segmentation
\item \doxylink{group__pose_net}{pose\+Net} for pose estimation
\item \doxylink{group__depth_net}{depth\+Net} for mono depth
\end{DoxyItemize}

\doxylink{requirements_The}{The} different primitives each inherit from the shared \doxylink{group__tensor_net}{tensor\+Net} object.
