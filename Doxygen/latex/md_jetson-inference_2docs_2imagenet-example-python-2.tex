In the previous step, we ran a sample application that came with the {\ttfamily jetson-\/inference} repo. ~\newline


Now, we\textquotesingle{}re going to walk through creating a new program from scratch in Python for image recognition called \href{../python/examples/my-recognition.py}{\texttt{{\ttfamily my-\/recognition.\+py}}}. This script will load an arbitrary image from disk and classify it using the \href{https://rawgit.com/dusty-nv/jetson-inference/master/docs/html/python/jetson.inference.html\#imageNet}{\texttt{{\ttfamily image\+Net}}} object. \doxylink{requirements_The}{The} completed source at \href{../python/examples/my-recognition.py}{\texttt{{\ttfamily python/\+examples/\+my-\/recognition.\+py}}}

\DoxyCodeLine{}

If you\textquotesingle{}re using the Docker container, you\textquotesingle{}ll want to store your code in a \href{aux-docker.md\#mounted-data-volumes}{\texttt{Mounted Directory}}. This way your code won\textquotesingle{}t be lost when you shutdown the container. For simplicity, this guide will create it in a directory on your host device under the user\textquotesingle{}s home directory located at {\ttfamily \texorpdfstring{$\sim$}{\string~}/\+my-\/recognition-\/python}, and then mount that path into the container.

See the \href{imagenet-console-2.md\#downloading-other-classification-models}{\texttt{Downloading Other Classification Models}} section from the previous page for more information about downloading other networks.\hypertarget{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md295}{}\doxysubsubsubsection{\texorpdfstring{Loading the Image from Disk}{Loading the Image from Disk}}\label{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md295}
You can load images from disk into shared CPU/\+\+GPU memory using the {\ttfamily \doxylink{group__image_gab2959819259e3cbcfd72ad633f8f9f17}{load\+Image()}} function. Supported formats are JPG, PNG, TGA, and BMP.

For more information about accessing images from Python, see the \doxysectlink{md_jetson-inference_2docs_2aux-image}{Image Manipulation with CUDA}{0} page. For simplicity, we just load a single image here. To load a video or sequence of images, you would want to use the \href{aux-streaming.md\#source-code}{\texttt{{\ttfamily video\+Source}}} API like the previous \href{../python/examples/imagenet.py}{\texttt{{\ttfamily imagenet.\+py}}} sample does.\hypertarget{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md296}{}\doxysubsubsubsection{\texorpdfstring{Loading the Image Recognition Network}{Loading the Image Recognition Network}}\label{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md296}
Using the \href{https://rawgit.com/dusty-nv/jetson-inference/master/docs/html/python/jetson.inference.html\#imageNet}{\texttt{{\ttfamily image\+Net}}} object, the following code will load the desired classification model with Tensor\+RT. Unless you specified a different network using the {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag, by default it will load Google\+Net, which was already downloaded when you initially \href{building-repo-2.md\#downloading-models}{\texttt{built the {\ttfamily jetson-\/inference} repo}} (the {\ttfamily Res\+Net-\/18} model was also selected by default to be downloaded).

All of the available classification models are pre-\/trained on the Image\+Net ILSVRC dataset, which can recognize up to \href{../data/networks/ilsvrc12_synset_words.txt}{\texttt{1000 different classes}} of objects, like different kinds of fruits and vegetables, many different species of animals, along with everyday man-\/made objects like vehicles, office furniture, sporting equipment, ect. ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{\ python}

\DoxyCodeLine{net\ =\ jetson.inference.imageNet(args.network)}

\end{DoxyCode}
\hypertarget{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md297}{}\doxysubsubsubsection{\texorpdfstring{Classifying the Image}{Classifying the Image}}\label{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md297}
Next, we are going to classify the image with the recognition network using the {\ttfamily \doxylink{classimage_net_a0023fb5bcd45dcefdac6a7df4bf9bab5}{image\+Net.\+Classify()}} function:\+ ~\newline



\begin{DoxyCode}{0}
\DoxyCodeLine{\ python}
\DoxyCodeLine{\#\ classify\ the\ image}
\DoxyCodeLine{class\_idx,\ confidence\ =\ net.Classify(img)}

\end{DoxyCode}


{\ttfamily \doxylink{classimage_net_a0023fb5bcd45dcefdac6a7df4bf9bab5}{image\+Net.\+Classify()}} accepts the image and it\textquotesingle{}s dimensions, and performs the inferencing with Tensor\+RT. ~\newline


It returns a tuple containing the integer index of the object class that the image was recognized as, along with the floating-\/point confidence value of the result.\hypertarget{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md298}{}\doxysubsubsubsection{\texorpdfstring{Interpreting the Results}{Interpreting the Results}}\label{md_jetson-inference_2docs_2imagenet-example-python-2_autotoc_md298}
As the final step, let\textquotesingle{}s retrieve the class description and print out the results of the classification:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{\ python}
\DoxyCodeLine{\#\ find\ the\ object\ description}
\DoxyCodeLine{class\_desc\ =\ net.GetClassDesc(class\_idx)}
\DoxyCodeLine{}
\DoxyCodeLine{\#\ print\ out\ the\ result}
\DoxyCodeLine{print("{}image\ is\ recognized\ as\ '\{:s\}'\ (class\ \#\{:d\})\ with\ \{:f\}\%\ confidence"{}.format(class\_desc,\ class\_idx,\ confidence\ *\ 100))}

\end{DoxyCode}


{\ttfamily \doxylink{classimage_net_a0023fb5bcd45dcefdac6a7df4bf9bab5}{image\+Net.\+Classify()}} returns the index of the recognized object class (between {\ttfamily 0} and {\ttfamily 999} for these models that were trained on ILSVRC). Given the class index, the {\ttfamily \doxylink{classimage_net_a9bfc031d94c241692f1dbbdadb0fed1f}{image\+Net.\+Get\+Class\+Desc()}} function will then return the string containing the text description of that class. These descriptions are automatically loaded from \href{../data/networks/ilsvrc12_synset_words.txt}{\texttt{{\ttfamily ilsvrc12\+\_\+synset\+\_\+words.\+txt}}}.
