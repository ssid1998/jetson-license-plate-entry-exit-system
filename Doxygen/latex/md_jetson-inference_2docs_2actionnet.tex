 \textsuperscript{Action Recognition}\hypertarget{md_jetson-inference_2docs_2actionnet_autotoc_md42}{}\doxysection{\texorpdfstring{Action Recognition}{Action Recognition}}\label{md_jetson-inference_2docs_2actionnet_autotoc_md42}
Action recognition classifies the activity, behavior, or gesture occuring over a sequence of video frames. \doxylink{requirements_The}{The} DNNs typically use image classification backbones with an added temporal dimension. For example, the Res\+Net18-\/based pre-\/trained models use a window of 16 frames. You can also skip frames to lengthen the window of time over which the model classifies actions.



\doxylink{requirements_The}{The} \href{../c/actionNet.h}{\texttt{{\ttfamily action\+Net}}} object takes in one video frame at a time, buffers them as input to the model, and outputs the class with the highest confidence. \href{../c/actionNet.h}{\texttt{{\ttfamily action\+Net}}} can be used from \href{https://rawgit.com/dusty-nv/jetson-inference/master/docs/html/python/jetson.inference.html\#actionNet}{\texttt{Python}} and \href{../c/actionNet.h}{\texttt{C++}}.

As examples of using the {\ttfamily \doxylink{classaction_net}{action\+Net}} class, there are sample programs for C++ and Python:\+


\begin{DoxyItemize}
\item \href{../examples/actionnet/actionnet.cpp}{\texttt{{\ttfamily actionnet.\+cpp}}} (C++)
\item \href{../python/examples/actionnet.py}{\texttt{{\ttfamily actionnet.\+py}}} (Python)
\end{DoxyItemize}\hypertarget{md_jetson-inference_2docs_2actionnet_autotoc_md43}{}\doxysubsection{\texorpdfstring{Running the Example}{Running the Example}}\label{md_jetson-inference_2docs_2actionnet_autotoc_md43}

By default, the model will process every-\/other frame to lengthen the window of time for classifying actions over. You can change this with the {\ttfamily -\/-\/skip-\/frames} parameter (using {\ttfamily -\/-\/skip-\/frames=0} will process every frame).\hypertarget{md_jetson-inference_2docs_2actionnet_autotoc_md45}{}\doxysubsubsection{\texorpdfstring{Pre-\/trained Action Recognition Models}{Pre-\/trained Action Recognition Models}}\label{md_jetson-inference_2docs_2actionnet_autotoc_md45}
