\doxysubsection{Detailed Description}
Hardware-\/accelerated video decoder for Jetson using GStreamer.

\doxylink{classgst_decoder}{gst\+Decoder} supports loading video files from disk (MKV, MP4, AVI, FLV) and RTP/\+\+RTSP network streams over UDP/\+\+IP. \doxylink{requirements_The}{The} supported decoder codecs are H.\+264, H.\+265, VP8, VP9, MPEG-\/2, MPEG-\/4, and MJPEG.

\begin{DoxyNote}{Note}
\doxylink{classgst_decoder}{gst\+Decoder} implements the \doxylink{classvideo_source}{video\+Source} interface and is intended to be used through that as opposed to directly. \doxylink{classvideo_source}{video\+Source} implements additional command-\/line parsing of \doxylink{structvideo_options}{video\+Options} to construct instances.
\end{DoxyNote}
