class nlohmann::\+json\+\_\+pointer$<$ Basic\+Json\+Type $>$}
JSON Pointer. 

A JSON pointer defines a string syntax for identifying a specific value within a JSON document. It can be used with functions {\ttfamily at} and {\ttfamily operator\+[]\+}. Furthermore, JSON pointers are the base for JSON patches.
