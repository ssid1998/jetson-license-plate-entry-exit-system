 \textsuperscript{Semantic Segmentation}\hypertarget{md_jetson-inference_2docs_2segnet-console-2_autotoc_md396}{}\doxysection{\texorpdfstring{Semantic Segmentation with Seg\+Net}{Semantic Segmentation with Seg\+Net}}\label{md_jetson-inference_2docs_2segnet-console-2_autotoc_md396}
\doxylink{requirements_The}{The} next deep learning capability we\textquotesingle{}ll cover in this tutorial is {\bfseries{semantic segmentation}}. Semantic segmentation is based on image recognition, except the classifications occur at the pixel level as opposed to the entire image. This is accomplished by {\itshape convolutionalizing} a pre-\/trained image recognition backbone, which transforms the model into a \href{https://arxiv.org/abs/1605.06211}{\texttt{Fully Convolutional Network (FCN)}} capable of per-\/pixel labeling. Especially useful for environmental perception, segmentation yields dense per-\/pixel classifications of many different potential objects per scene, including scene foregrounds and backgrounds.



\href{../c/segNet.h}{\texttt{{\ttfamily seg\+Net}}} accepts as input the 2D image, and outputs a second image with the per-\/pixel classification mask overlay. Each pixel of the mask corresponds to the class of object that was classified. \href{../c/segNet.h}{\texttt{{\ttfamily seg\+Net}}} is available to use from \href{https://rawgit.com/dusty-nv/jetson-inference/master/docs/html/python/jetson.inference.html\#segNet}{\texttt{Python}} and \href{../c/segNet.h}{\texttt{C++}}. ~\newline


As examples of using the {\ttfamily \doxylink{classseg_net}{seg\+Net}} class, we provide sample programs C++ and Python:\+


\begin{DoxyItemize}
\item \href{../examples/segnet/segnet.cpp}{\texttt{{\ttfamily segnet.\+cpp}}} (C++)
\item \href{../python/examples/segnet.py}{\texttt{{\ttfamily segnet.\+py}}} (Python)
\end{DoxyItemize}

These samples are able to segment images, videos, and camera feeds. For more info about the various types of input/\+output streams supported, see the \doxysectlink{md_jetson-inference_2docs_2aux-streaming}{Camera Streaming and Multimedia}{0} page.

See below for various pre-\/trained segmentation models available that use the FCN-\/\+Res\+Net18 network with realtime performance on Jetson. First, let\textquotesingle{}s try using the {\ttfamily segnet} program to segment static images. In addition to the input/\+output paths, there are some additional command-\/line options:\+


\begin{DoxyItemize}
\item optional {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag changes the segmentation model being used (see above)
\item optional {\ttfamily -\/-\/visualize} flag accepts {\ttfamily mask} and/\+or {\ttfamily \doxylink{group__overlay}{Overlay}} modes (default is {\ttfamily \doxylink{group__overlay}{Overlay}})
\item optional {\ttfamily -\/-\/alpha} flag sets the alpha blending value for {\ttfamily \doxylink{group__overlay}{Overlay}} (default is {\ttfamily 120})
\item optional {\ttfamily -\/-\/filter-\/mode} flag accepts {\ttfamily point} or {\ttfamily linear} sampling (default is {\ttfamily linear})
\end{DoxyItemize}

Launch the application with the {\ttfamily -\/-\/help} flag for more info, and refer to the \doxysectlink{md_jetson-inference_2docs_2aux-streaming}{Camera Streaming and Multimedia}{0} page for supported input/\+output protocols.
