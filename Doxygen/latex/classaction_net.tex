Set the number of frames that are skipped in between classifications. Since \doxylink{classaction_net}{action\+Net} operates on video sequences, it\textquotesingle{}s often helpful to skip frames to lengthen the window of time the model gets to \textquotesingle{}see\textquotesingle{} an action being performed.

\doxylink{requirements_The}{The} default setting is 1, where every other frame is skipped. Setting this to 0 will disable it, and every frame will be processed. When a frame is skipped, the classification results from the last frame are returned. \Hypertarget{classaction_net_a9628498f941ed6709c6b2e13bbb79e29}\index{actionNet@{actionNet}!SetThreshold@{SetThreshold}}
