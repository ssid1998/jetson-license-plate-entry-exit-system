\hypertarget{md_jetson-inference_2docs_2digits-setup_autotoc_md239}{}\doxysection{\texorpdfstring{DIGITS System Setup}{DIGITS System Setup}}\label{md_jetson-inference_2docs_2digits-setup_autotoc_md239}
During this tutorial, we\textquotesingle{}ll use a host PC (or cloud instance) for training DNNs, alongside a Jetson for inference. ~\newline


Due to the number of dependencies required for training, it\textquotesingle{}s recommended for beginners to setup their host training PC with {\bfseries{\href{https://www.nvidia.com/en-us/gpu-cloud/}{\texttt{NVIDIA GPU Cloud (NGC)}}}} or \href{https://github.com/NVIDIA/nvidia-docker}{\texttt{nvidia-\/docker}}. These methods automate the install of the drivers and machine learning frameworks on the host. NGC can be used to deploy Docker images locally, or remotely to cloud providers like AWS or Azure N-\/series.

A host PC will also serve to flash the Jetson with the latest Jet\+Pack. First, we\textquotesingle{}ll setup and configure the host training PC with the required OS and tools.\hypertarget{md_jetson-inference_2docs_2digits-setup_autotoc_md240}{}\doxysubsubsection{\texorpdfstring{Installing Ubuntu on the Host}{Installing Ubuntu on the Host}}\label{md_jetson-inference_2docs_2digits-setup_autotoc_md240}
If you don\textquotesingle{}t already have Ubuntu installed on your host PC, download and install Ubuntu 16.\+04 x86\+\_\+64 from one of the following locations:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{http://releases.ubuntu.com/16.04/ubuntu-\/16.04.2-\/desktop-\/amd64.iso}
\DoxyCodeLine{http://releases.ubuntu.com/16.04/ubuntu-\/16.04.2-\/desktop-\/amd64.iso.torrent}

\end{DoxyCode}


Ubuntu 14.\+04 x86\+\_\+64 or Ubuntu 18.\+04 x86\+\_\+64 may also be acceptable with minor modifications later while installing some packages with apt-\/get.\hypertarget{md_jetson-inference_2docs_2digits-setup_autotoc_md241}{}\doxysubsubsection{\texorpdfstring{Setting up host training PC with NGC container}{Setting up host training PC with NGC container}}\label{md_jetson-inference_2docs_2digits-setup_autotoc_md241}
\begin{quote}
{\bfseries{note}}:\+ to setup DIGITS natively on your host PC, you should go to \doxysectlink{md_jetson-inference_2docs_2digits-native}{{\ttfamily Natively setting up DIGITS on the Host}}{0} (advanced) ~\newline
 \end{quote}


NVIDIA hosts NVIDIA® GPU Cloud (NGC) container registry for AI developers worldwide. You can download a containerized software stack for a wide range of deep learning frameworks, optimized and verified with NVIDIA libraries and CUDA runtime version.



If you have a recent generation GPU (Pascal or newer) on your PC, the use of NGC registry container is probably the easiest way to setup DIGITS. To use a NGC registry container on your local host machine (as opposed to cloud), you can follow this detailed \href{https://docs.nvidia.com/ngc/ngc-titan-setup-guide/index.html}{\texttt{setup guide}}.\hypertarget{md_jetson-inference_2docs_2digits-setup_autotoc_md242}{}\doxysubsubsubsection{\texorpdfstring{Installing the NVIDIA driver}{Installing the NVIDIA driver}}\label{md_jetson-inference_2docs_2digits-setup_autotoc_md242}
