\doxysubsubsection{Detailed Description}
DNN abstract base class that provides Tensor\+RT functionality underneath. These functions aren\textquotesingle{}t typically accessed by end users unless they are implementing their own DNN class like \doxylink{classimage_net}{image\+Net} or \doxylink{classdetect_net}{detect\+Net}.
