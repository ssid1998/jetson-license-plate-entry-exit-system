 \textsuperscript{Object Detection}\hypertarget{md_jetson-inference_2docs_2detectnet-tracking_autotoc_md217}{}\doxysection{\texorpdfstring{Object Tracking on Video}{Object Tracking on Video}}\label{md_jetson-inference_2docs_2detectnet-tracking_autotoc_md217}
Although with the accuracy of modern detection DNNs you can essentially do "{}tracking by detection"{}, some degree of temporal filtering can be beneficial to smooth over blips in the detections and temporary occlusions in the video. jetson-\/inference includes basic (but fast) multi-\/object tracking using frame-\/to-\/frame IOU (intersection-\/over-\/union) bounding box comparisons from \href{http://elvera.nue.tu-berlin.de/typo3/files/1517Bochinski2017.pdf}{\texttt{{\ttfamily High-\/\+Speed Tracking-\/by-\/\+Detection Without Using Image Information}}} (Deep\+Stream has more comprehensive tracking implementations available \href{https://docs.nvidia.com/metropolis/deepstream/dev-guide/text/DS_plugin_gst-nvtracker.html}{\texttt{from here}}).

\href{https://www.youtube.com/watch?v=L8vwuXKQrow}{\texttt{}}

To enable tracking with detectnet/\+detectnet.\+py, run it with the {\ttfamily -\/-\/tracking} flag. ~\newline
