\doxysubsection{Detailed Description}
\doxylink{class_thread}{Thread} class for launching an asynchronous operating-\/system dependent thread. To make your own thread, provide a function pointer of the thread\textquotesingle{}s entry point, or inherit from this class and implement your own \doxylink{class_thread_a26c650bc1d2d290979a2f693569ab9c8}{Run()} function.
