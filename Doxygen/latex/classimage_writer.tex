\doxysubsection{Detailed Description}
Save an image or set of images to disk.

Supported image formats for saving are JPG, PNG, TGA, and BMP. Internally, \doxylink{classimage_loader}{image\+Loader} uses the \doxylink{group__image_gaebd1293bd7119be5164b521dc20b2233}{save\+Image()} function to save the images, so the supported formats are the same.

\doxylink{classimage_writer}{image\+Writer} has the ability to write a sequence of images to a directory, for example {\ttfamily images/\+i.\+jpg} (where {\ttfamily i} becomes the image number), or just a single image with a static filename (e.\+g. {\ttfamily images/\+my\+\_\+image.\+jpg}). When given just the path of a directory as output, it will default to incremental {\ttfamily i.\+jpg} sequencing and save in JPG format.

\begin{DoxyNote}{Note}
\doxylink{classimage_writer}{image\+Writer} implements the \doxylink{classvideo_output}{video\+Output} interface and is intended to be used through that as opposed to directly. \doxylink{classvideo_output}{video\+Output} implements additional command-\/line parsing of \doxylink{structvideo_options}{video\+Options} to construct instances.
\end{DoxyNote}
