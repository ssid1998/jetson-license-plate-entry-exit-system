\doxysubsection{Detailed Description}
\doxylink{classgst_buffer_manager}{gst\+Buffer\+Manager} recieves GStreamer buffers from appsink elements and unpacks/\+maps them into CUDA address space, and handles colorspace conversion into RGB format.

It can handle both normal CPU-\/based GStreamer buffers and NVMM memory which can be mapped directly to the GPU without requiring memory copies using the CPU.

To disable the use of NVMM memory, set -\/DENABLE\+\_\+\+NVMM=OFF when building with CMake:\+ \begin{DoxyVerb}cmake -DENABLE_NVMM=OFF ../
\end{DoxyVerb}
