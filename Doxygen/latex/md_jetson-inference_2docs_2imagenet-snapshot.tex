Now that we confirmed the trained model is working in DIGITS, let\textquotesingle{}s download and extract the model snapshot to Jetson. ~\newline


From the browser on your Jetson, navigate to your DIGITS server and the {\ttfamily Google\+Net-\/\+ILSVRC12-\/subset} model. Under the {\ttfamily Trained Models} section, select the desired snapshot from the drop-\/down (usually the one with the highest epoch) and click the {\ttfamily Download Model} button.



Alternatively, if your Jetson and DIGITS server aren\textquotesingle{}t accessible from the same network, you can use the step above to download the snapshot to an intermediary machine and then use SCP or USB stick to copy it to Jetson. ~\newline


Then extract the archive with a command similar to:\+

\DoxyCodeLine{tar\ -\/xzvf\ 20170524-\/140310-\/8c0b\_epoch\_30.0.tar.gz}

\end{DoxyCode}
