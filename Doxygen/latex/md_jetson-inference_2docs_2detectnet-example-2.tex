\doxylink{requirements_The}{The} completed source is available in the \href{../python/examples/my-detection.py}{\texttt{{\ttfamily python/\+examples/\+my-\/detection.\+py}}} file of the repo, but the guide below will act like they reside in the user\textquotesingle{}s home directory or in an arbitrary directory of your choosing. Next the detection network processes the image with the {\ttfamily net.\+Detect()} function. It takes in the image from {\ttfamily camera.\+Capture()} and returns a list of detections:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{python}
\DoxyCodeLine{\ \ \ detections\ =\ net.Detect(img)}

\end{DoxyCode}


This function will also automatically overlay the detection results on top of the input image.

If you want, you can add a {\ttfamily print(detections)} statement here, and the coordinates, confidence, and class info will be printed out to the terminal for each detection result. Also see the \href{https://rawgit.com/dusty-nv/jetson-inference/master/docs/html/python/jetson.inference.html\#detectNet}{\texttt{{\ttfamily detect\+Net}}} documentation for info about the different members of the {\ttfamily Detection} structures that are returned for accessing them directly in a custom application.\hypertarget{md_jetson-inference_2docs_2detectnet-example-2_autotoc_md206}{}\doxysubsubsubsection{\texorpdfstring{Rendering}{Rendering}}\label{md_jetson-inference_2docs_2detectnet-example-2_autotoc_md206}
