\doxysubsection{Detailed Description}
Open\+GL display window and image/\+video renderer with CUDA interoperability.

\begin{DoxyNote}{Note}
\doxylink{classgl_display}{gl\+Display} implements the \doxylink{classvideo_output}{video\+Output} interface and is intended to be used through that as opposed to directly. \doxylink{classvideo_output}{video\+Output} implements additional command-\/line parsing of \doxylink{structvideo_options}{video\+Options} to construct instances.
\end{DoxyNote}
