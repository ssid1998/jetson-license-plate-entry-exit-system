\hypertarget{md_jetson-inference_2docs_2imagenet-console_autotoc_md265}{}\doxysection{\texorpdfstring{Classifying Images with Image\+Net}{Classifying Images with Image\+Net}}\label{md_jetson-inference_2docs_2imagenet-console_autotoc_md265}
There are multiple types of deep learning networks available, including recognition, detection/\+localization, and semantic segmentation. \doxylink{requirements_The}{The} first deep learning capability we\textquotesingle{}re highlighting in this tutorial is {\bfseries{image recognition}} using classifcation networks that have been trained to identify scenes and objects.

\doxylink{requirements_The}{The} \href{../c/imageNet.h}{\texttt{{\ttfamily image\+Net}}} object accepts an input image and outputs the probability for each class. Having been trained on the Image\+Net ILSVRC dataset of {\bfseries{\href{../data/networks/ilsvrc12_synset_words.txt}{\texttt{1000 objects}}}}, the Google\+Net and Res\+Net-\/18 models were automatically downloaded during the build step. See below for other classification models that can be downloaded and used as well.

As examples of using \href{../c/imageNet.h}{\texttt{{\ttfamily image\+Net}}} we provide versions of a command-\/line interface for C++ and Python:\+


\begin{DoxyItemize}

\item \href{../python/examples/imagenet-console.py}{\texttt{{\ttfamily imagenet-\/console.\+py}}} (Python)
\end{DoxyItemize}

Later in the tutorial, we\textquotesingle{}ll also cover versions of a live camera recognition program for C++ and Python:\+

\end{DoxyItemize}\hypertarget{md_jetson-inference_2docs_2imagenet-console_autotoc_md266}{}\doxysubsubsection{\texorpdfstring{Using the Console Program on Jetson}{Using the Console Program on Jetson}}\label{md_jetson-inference_2docs_2imagenet-console_autotoc_md266}
First, let\textquotesingle{}s try using the {\ttfamily imagenet-\/console} program to test \doxylink{classimage_net}{image\+Net} recognition on some example images. It loads an image, uses Tensor\+RT and the {\ttfamily \doxylink{classimage_net}{image\+Net}} class to perform the inference, then overlays the classification result and saves the output image. \doxylink{requirements_The}{The} repo comes with some sample images for you to use.

After \doxysectlink{md_jetson-inference_2docs_2building-repo-2}{building}{0} the repo, make sure your terminal is located in the {\ttfamily aarch64/\+bin} directory:\+


\begin{DoxyCode}{0}
\DoxyCodeLine{\ bash}
\DoxyCodeLine{\$\ cd\ jetson-\/inference/build/aarch64/bin}

\end{DoxyCode}


Next, let\textquotesingle{}s classify an example image with the {\ttfamily imagenet-\/console} program, using either the \href{../examples/imagenet-console/imagenet-console.cpp}{\texttt{C++}} or \href{../python/examples/imagenet-console.py}{\texttt{Python}} variants. ~\newline


{\ttfamily imagenet-\/console} accepts 3 command-\/line arguments:\+ ~\newline



\begin{DoxyItemize}
\item the path to an input image ({\ttfamily jpg, png, tga, bmp})
\item optional path to output image ({\ttfamily jpg, png, tga, bmp})
\item optional {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag which changes the classification model being used (the default network is Google\+Net). ~\newline

\end{DoxyItemize}

Note that there are additional command line parameters available for loading customized models. Launch the application with the {\ttfamily -\/-\/help} flag to recieve more info about using them, or see the \href{../README.md\#code-examples}{\texttt{{\ttfamily Code Examples}}} readme.

Generally the more complex networks can have greater classification accuracy, with increased runtime.\hypertarget{md_jetson-inference_2docs_2imagenet-console_autotoc_md272}{}\doxysubsubsection{\texorpdfstring{Using Different Classification Models}{Using Different Classification Models}}\label{md_jetson-inference_2docs_2imagenet-console_autotoc_md272}
You can specify which model to load by setting the {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag on the command line to one of the corresponding CLI arguments from the table above. By default, Google\+Net is loaded if the optional {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag isn\textquotesingle{}t specified.
