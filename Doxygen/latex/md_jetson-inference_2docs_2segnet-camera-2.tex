\begin{DoxyItemize}
\item MIPI CSI cameras ({\ttfamily csi:\+/\+/\+0})
\item V4\+L2 cameras ({\ttfamily /\+dev/\+video0})
\item RTP/\+\+RTSP streams ({\ttfamily rtsp:\+/\+/\+username:\+password@ip:\+port})
\end{DoxyItemize}

For more information about video streams and protocols, please see the \doxysectlink{md_jetson-inference_2docs_2aux-streaming}{Camera Streaming and Multimedia}{0} page.

Run the program with {\ttfamily -\/-\/help} to see a full list of options -\/ some of them specific to \doxylink{classseg_net}{seg\+Net} include:\+


\begin{DoxyItemize}
\item optional {\ttfamily -\/-\/\doxylink{group__network}{Networking}} flag changes the segmentation model being used (see \href{segnet-console-2.md\#pre-trained-segmentation-models-available}{\texttt{available networks}})
\item optional {\ttfamily -\/-\/visualize} flag accepts {\ttfamily mask} and/\+or {\ttfamily \doxylink{group__overlay}{Overlay}} modes (default is {\ttfamily \doxylink{group__overlay}{Overlay}})
\item optional {\ttfamily -\/-\/alpha} flag sets the alpha blending value for the overlay (default is {\ttfamily 120})
\item optional {\ttfamily -\/-\/filter-\/mode} flag accepts {\ttfamily point} or {\ttfamily linear} sampling (default is {\ttfamily linear})
\end{DoxyItemize}
