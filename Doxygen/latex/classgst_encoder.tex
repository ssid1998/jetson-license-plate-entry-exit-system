\doxysubsection{Detailed Description}
Hardware-\/accelerated video encoder for Jetson using GStreamer.

\doxylink{requirements_The}{The} encoder can write the encoded video to disk in (MKV, MP4, AVI, FLV), or stream over the network to a remote host via RTP/\+\+RTSP using UDP/\+\+IP. \doxylink{requirements_The}{The} supported encoder codecs are H.\+264, H.\+265, VP8, VP9, and MJPEG.

\begin{DoxyNote}{Note}
\doxylink{classgst_encoder}{gst\+Encoder} implements the \doxylink{classvideo_output}{video\+Output} interface and is intended to be used through that as opposed to directly. \doxylink{classvideo_output}{video\+Output} implements additional command-\/line parsing of \doxylink{structvideo_options}{video\+Options} to construct instances.
\end{DoxyNote}
