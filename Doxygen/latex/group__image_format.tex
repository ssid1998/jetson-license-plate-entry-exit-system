\doxylink{requirements_The}{The} \doxylink{group__image_format_ga931c48e08f361637d093355d64583406}{image\+Format} enum is used to identify the pixel format and colorspace of an image. Supported data types are based on {\ttfamily uint8} and {\ttfamily float}, with colorspaces including RGB/\+\+RGBA, BGR/\+\+BGRA, grayscale, YUV, and Bayer.

There are also a variety of helper functions available that provide info about each format at runtime -\/ for example, the pixel bit depth (\doxylink{group__image_format_ga48d21d2cc2f8822e13ee6d40e3719ba4}{image\+Format\+Depth()}) the number of image channels (\doxylink{group__image_format_ga95cdbe993bce3c036d4941f2c3913f74}{image\+Format\+Channels()}), and computing the size of an image from it\textquotesingle{}s dimensions (\begin{DoxySeeAlso}{See also}
\doxylink{group__image_format_gae8779cdbde4c4543a26a736ef7222fae}{image\+Format\+Size()}). To convert between \doxylink{group__image}{Image I/\+O} formats using the GPU, there is also the \doxylink{group__colorspace_ga300b4f0ab2845de6e13dafc88facdb2f}{cuda\+Convert\+Color()} function.
\end{DoxySeeAlso}

\doxylink{requirements_The}{The} \doxylink{group__image_format_ga931c48e08f361637d093355d64583406}{image\+Format} enum is used to identify the pixel format and colorspace of an image. Supported data types are based on {\ttfamily uint8} and {\ttfamily float}, with colorspaces including RGB/\+\+RGBA, BGR/\+\+BGRA, grayscale, YUV, and Bayer.

There are also a variety of helper functions available that provide info about each format at runtime -\/ for example, the pixel bit depth (\doxylink{group__image_format_ga48d21d2cc2f8822e13ee6d40e3719ba4}{image\+Format\+Depth()}) the number of image channels (\doxylink{group__image_format_ga95cdbe993bce3c036d4941f2c3913f74}{image\+Format\+Channels()}), and computing the size of an image from it\textquotesingle{}s dimensions (\begin{DoxySeeAlso}{See also}
\doxylink{group__image_format_gae8779cdbde4c4543a26a736ef7222fae}{image\+Format\+Size()}). To convert between \doxylink{group__image}{Image I/\+O} formats using the GPU, there is also the \doxylink{group__colorspace_ga300b4f0ab2845de6e13dafc88facdb2f}{cuda\+Convert\+Color()} function.
\end{DoxySeeAlso}
