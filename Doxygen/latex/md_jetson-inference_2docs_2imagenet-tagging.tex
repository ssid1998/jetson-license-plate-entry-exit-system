Multi-\/label classification models are able to recognize multiple object classes simultaneously for performing tasks like image tagging. \doxylink{requirements_The}{The} multi-\/label DNNs are almost identical in topology to ordinary single-\/class models, except they use a sigmoid activation layer as opposed to softmax. There\textquotesingle{}s a pre-\/trained {\ttfamily resnet18-\/tagging-\/voc} multi-\/label model available that was trained on the Pascal VOC dataset:\+ ~\newline
