	\textbf{Project: Number Plate Detection using Jetson Nano} \\
	Edge deployment on NVIDIA Jetson Nano using YOLOv8-N (Nano) and YOLOv8-L (Large) models; local SQL database logging and dwell-time based billing.
\end{center}

\vspace{0.5cm}

\section{Executive Summary}
This project proposes a **single-camera license plate recognition (LPR) and billing system** for German airport and security gate scenarios. It combines:
\begin{itemize}
	\item \textbf{YOLOv8 models} for fast, robust plate detection \cite{jocher2023ultralytics}.
	\item \textbf{OCR engines} such as EasyOCR/Tesseract for plate text extraction.
	\item A lightweight \textbf{SQLite database} for local logging, dwell-time calculation, and billing.
	\item Built-in handling of \textbf{exceptions} (e.g., police, emergency, diplomatic vehicles).
\end{itemize}

The system emphasizes real-time performance, scalability, and GDPR-compliance by performing all processing on edge hardware (Jetson Nano) without reliance on cloud servers.

\section{Motivation and Relevance}
European airports face high throughput and strict security/privacy requirements:
\begin{itemize}
	\item Manual verification is slow and costly.
	\item Proprietary ANPR cameras are expensive and vendor-locked.
	\item YOLO-based edge systems offer flexibility, cost savings, and fast local decisions.
\end{itemize}

\section{Key Objectives}
\begin{enumerate}
	\item Develop a real-time YOLOv8-Nano detection pipeline for German license plates.
	\item Integrate OCR for text extraction, including suffixes (E, H) and color-coded diplomatic/military plates.
	\item Log events in a GDPR-compliant local SQLite database.
	\item Implement billing logic with dwell-time calculation and exemptions for special plates.
	\item Validate under varied conditions (lighting, weather, skewed angles).
\end{enumerate}

\section{Contribution}
This chapter establishes:
\begin{itemize}
	\item The \textbf{technical motivation} for YOLO-based edge ANPR systems.
	\item A structured overview of the system’s pipeline (detection → OCR → database → billing).
	\item The legal and operational relevance of deploying ANPR systems in German contexts.
\end{itemize}
