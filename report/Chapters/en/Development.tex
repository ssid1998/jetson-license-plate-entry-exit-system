\chapter{Development folder structures}


\section{Folder Structure}

The project's directory is systematically organized to facilitate easy navigation and comprehension. Key components include:

\begin{itemize}
	\item \texttt{.ipynb\_checkpoints/}: Contains backups of Jupyter notebooks for version control.
	\item \texttt{anprsys/}: Houses the main application scripts, TensorFlow models, and environmental setup files.
	\item \texttt{Tensorflow/}: Dedicated directory for TensorFlow resources, models, and scripts.
	\item \texttt{.gitignore}: Git configuration file for untracked files.
	\item \texttt{README.md}: Documentation providing essential information about the project.
	\item \texttt{report.txt}: Contains generated reports or logs from the project's operation.
	\item \texttt{Error Guide.md}: A markdown guide detailing troubleshooting steps for common errors.
	\item Jupyter Notebooks: Interactive notebooks like \texttt{Automatic Number Plate Detection.ipynb} for executing the object detection pipeline.
	\item \texttt{Detection\_Images/}: Stores the result images of the license plates detected.
	\item \texttt{detection\_results.csv}: Stores the final correct detected license plate along with UUID.
	\item \texttt{realtimeresults.csv}: Stores the detected license plate via the Logitech webcam along with UUID in real-time.
\end{itemize}

\section{Overview}

Operational execution of the project requires:

\begin{itemize}
	\item Python programming environment.
	\item TensorFlow framework for machine learning tasks.
	\item Supporting libraries like OpenCV and easyOCR for image processing and character recognition.
	\item The specific libraries and their versions are listed in the \texttt{requirements.txt} file.
\end{itemize}

\section{Installation}

Installation involves:

\begin{enumerate}
	\item Establishing a Python virtual environment.
	\item Installing necessary libraries using pip, guided by the \texttt{requirements.txt} file.
	\item Configuring TensorFlow and related models for object detection, acquired from TensorFlow's official repository or Model Zoo.
\end{enumerate}

\section{First Steps}

Post-installation, the initial steps include:

\begin{enumerate}
	\item Activating the Python virtual environment.
	\item Commencing the interactive development process via provided Jupyter notebooks, beginning with image data collection and proceeding to model training and object detection.
\end{enumerate}

\section{Example}

A practical example of the project's application would be:

\begin{enumerate}
	\item Executing the \texttt{1. Image Collection.ipynb} notebook to acquire training data.
	\item Running the \texttt{2. Training and Detection.ipynb} notebook to train the model.
	\item Utilizing the trained model to detect and recognize number plates in new visual data sets.
\end{enumerate}
