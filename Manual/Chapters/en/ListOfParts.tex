\chapter{List Of Parts}

The following is a list of components included in the box with the Jetson Nano J1010, as well as accessories required but not included for initial setup:

\section*{Included in the Box}
\begin{itemize}[label=--, left=1.5em]
	\item reComputer J1010, including:
	\begin{itemize}[label=$\bullet$, left=2.5em]
		\item 4G Jetson Nano module x1
		\item J1010 carrier board x1
	\end{itemize}
\end{itemize}

\section*{Required Accessories (Not Included)}
\begin{itemize}[label=--, left=1.5em]
	\item USB Keyboard and mouse
	\item Display screen
	\item Type-C power cable and power supply
\end{itemize}

Ensure that all parts are present before starting the setup process. You will need to source the required accessories separately to power on and configure your Jetson Nano system.
