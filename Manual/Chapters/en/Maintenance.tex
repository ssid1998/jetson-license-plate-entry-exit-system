\chapter{Maintenance}

\section{Maintenance}

Regular maintenance keeps the \textbf{License Plate Detection System} running smoothly and extends its lifetime.  
This section describes how users can care for the hardware and software to ensure reliable daily operation at the airport gate.

\begin{tcolorbox}[colback=blue!5!white,colframe=blue!60!black,
	title={\hspace{0.5em}Important}, fonttitle=\bfseries]
	\begin{itemize}
		\item Turn off power before cleaning or adjusting cables.  
		\item Avoid using sharp objects or sprays near the camera lens.  
		\item Do not block the ventilation openings of the main unit's case.  
		\item If the device overheats or smells unusual, disconnect power immediately and contact support.
	\end{itemize}
\end{tcolorbox}


% ==========================================================
\subsection{Hardware Maintenance}

\begin{itemize}
	\item \textbf{Keep it clean:}  
	Gently wipe the main unit and camera with a dry microfiber cloth.  
	Avoid liquids, direct sunlight, or dust build-up around the device.
	
	\item \textbf{Check connections:}  
	Ensure all USB, HDMI, and power cables are firmly attached.  
	Loose connections can cause flickering displays or camera interruptions.
	
	\item \textbf{Maintain good cooling:}  
	Place the main unit  in a ventilated area.  
	If the device feels unusually hot, turn it off for a few minutes or use a small desk fan.
	
	\item \textbf{Power stability:}  
	Always use the original 5V/4A USB-C power adapter.  
	Unstable power may lead to sudden restarts or data loss.
	
	\item \textbf{Camera alignment:}  
	Check every few weeks that the camera faces the correct lane and captures clear plates without obstruction.
\end{itemize}

\subsection{Software Maintenance}

\begin{itemize}
	\item \textbf{Regular check-ups:}  
	Restart the main unit  once every few days to clear memory and refresh processes.
	
	\item \textbf{Keep the main unit updated:}  
	If an update notification appears, allow the main unit to install recommended main unit updates.
	
	\item \textbf{Database management:}  
	Every few weeks, check that the detection log is not full.  
	If storage is low, export or delete older records safely.
	
	\item \textbf{Model refresh (for developers):}  
	Advanced users can retrain or update the detection model to improve recognition accuracy over time.
\end{itemize}

\subsection{Backup and Safety}

\begin{itemize}
	\item \textbf{Weekly backup:}  
	Copy the database file (\texttt{plates.db}) to a USB drive or computer for safekeeping.
	
	\item \textbf{Power off correctly:}  
	Always shut down the device through the desktop before unplugging power.
	
	\item \textbf{Avoid sudden shutdowns:}  
	Interrupting the main unit during detection can damage stored data.
	
	\item \textbf{Emergency reset:}  
	If the screen freezes, disconnect power, wait 10 seconds, and reconnect.
	\item \textbf{Check storage space:}  
	Ensure there is enough free space on the main unit's microSD card.  
	Low storage can slow down performance or prevent new data from being saved.
\end{itemize}
