\chapter{Set Up}

\section{Required Components}
To set up your Jetson Nano J1010, you will need:
\begin{itemize}[label=--,left=1.5em]
	\item NVIDIA Jetson Nano J1010 board
	\item MicroSD card (16GB or larger) with pre-installed JetPack SDK
	\item USB keyboard and mouse
	\item HDMI monitor
	\item 5V power supply (4A recommended for maximum performance)
\end{itemize}

\section{Installing the Operating System}
\begin{enumerate}
	\item Download the JetPack SDK from \href{https://developer.nvidia.com/embedded/jetpack}{NVIDIA JetPack SDK}.
	\item Flash the image to a MicroSD card using tools like Etcher.
	\item Insert the MicroSD card into the slot on the Jetson Nano J1010.
	\item Connect the HDMI monitor, keyboard, and mouse.
	\item Power on the device by connecting the 5V power supply.
\end{enumerate}
