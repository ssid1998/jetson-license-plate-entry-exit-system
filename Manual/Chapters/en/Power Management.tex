\chapter{Power Management}

\section{Power Modes}
The Jetson Nano J1010 supports different power modes to balance performance and efficiency:
\begin{itemize}[label=--,left=1.5em]
	\item Mode 0: Maximum performance (10W)
	\item Mode 1: Energy-saving mode (5W)
\end{itemize}

Switch between power modes using the following command:
\begin{verbatim}
	sudo nvpmodel -m <MODE>
\end{verbatim}
