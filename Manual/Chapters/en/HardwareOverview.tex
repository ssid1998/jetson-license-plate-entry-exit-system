\chapter{Hardware Overview}

\section{Specifications}
\begin{itemize}[label=--,left=1.5em]
	\item CPU: Quad-core ARM Cortex-A57
	\item GPU: 128-core NVIDIA Maxwell
	\item Memory: 4 GB 64-bit LPDDR4
	\item Storage: MicroSD 16 GB eMMC 5.1
	\item USB: 1*USB 3.0 type A, 2*USB 2.0 type A, 1*USB type C for device mode, 1*USB type C for 5V power input
	\item Display: 1*HDMI 2.0 type A
	\item Power Adapter: USB type-C 5V/3A
\end{itemize}

\section{Physical Dimensions}
The Jetson Nano J1010 has the following physical dimensions:
\begin{itemize}[label=--,left=1.5em]
	\item Dimensions: 100 mm x 80 mm x 29 mm
	\item 130 mm x 120 mm x 50 mm (with case)
	\item Weight: 100g
\end{itemize}
