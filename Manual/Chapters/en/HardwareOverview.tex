\section{System Requirements}
To ensure smooth operation of the Vehicle License Plate Detection System, please confirm that you have all necessary hardware and software components listed below.

\subsection*{Hardware Requirements}
\begin{itemize}[label=--, left=1.5em]
	\item NVIDIA Jetson Nano J1010
	\item Camera compatible with Jetson Nano
	\item LED indicator (optional but recommended)
	\item 5V/3A Type-C power adapter
	\item MicroSD card (32GB or larger, pre-flashed with JetPack SDK)
	\item USB Keyboard and mouse
	\item Display screen with HDMI support
\end{itemize}

\subsection*{Software Requirements}
\begin{itemize}[label=--, left=1.5em]
	\item JetPack SDK (latest version recommended)
	\item Vehicle license plate dataset (e.g. from Kaggle)
	\item Python 3 environment with required libraries:
	\begin{itemize}[label=$\bullet$, left=2.5em]
		\item TensorFlow or PyTorch (for deep learning model inference)
		\item OpenCV (for image processing)
		\item GPIO libraries (for controlling the LED indicator)
	\end{itemize}
\end{itemize}
