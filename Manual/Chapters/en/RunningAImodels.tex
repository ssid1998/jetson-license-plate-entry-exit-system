\chapter{Running AI Models}

\section{Installing TensorFlow and PyTorch}
To run AI models, install TensorFlow or PyTorch:
\begin{enumerate}
	\item Install TensorFlow:
	\begin{verbatim}
		sudo pip3 install tensorflow
	\end{verbatim}
	\item Install PyTorch:
	\begin{verbatim}
		sudo apt-get install libopenblas-base libopenmpi-dev
		sudo pip3 install torch torchvision
	\end{verbatim}
\end{enumerate}

\section{Running Pre-Trained Models}
You can run pre-trained AI models using NVIDIA's DeepStream or TensorRT libraries:
\begin{itemize}[label=--,left=1.5em]
	\item Use DeepStream for video analytics and object detection.
	\item Use TensorRT for optimized, high-performance inference.
\end{itemize}
