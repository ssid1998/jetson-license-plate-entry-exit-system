			\block{Problem Description}
			{
				Efficient and automated vehicle identification has become essential in smart transportation, particularly for airport parking. This project focuses on designing
				single-camera vehicle license plate recognition system using Nvidia Jetson Nano.\newline \newline
				The system enables entry/exit license plate recognition and computes a fare based on the difference between entry and exit timestamps. Its design demonstrates the convergence of Artificial Intelligence (AI),
				embedded computing, and privacy-aware data management within modern transportation infrastructure.\newline
				
				There are multiple challenges while managing operations in places where vehicular traffic is high, such as airports, bus-stops and logistics parks. Manual verification thus becomes prone to errors, expends time, and has coherency issues, along with commercial cameras being expensive and having high
				maintenance. \newline
				
				Key issues include:\newline
				\begin{itemize}
					\item \textbf{High Throughput:} Thousands of vehicle entries per day demand sub-second recognition to prevent congestion.
					\item \textbf{Diverse Plate Formats:} Germany’s 15+ plate variants (electric, diplomatic, export, etc.) pose recognition challenges.
					\item \textbf{Environmental Variations:} Lighting changes, glare, and weather affect detection reliability. \newline
				\end{itemize}
				
				These constraints necessitate a locally deployed, energy-efficient, and privacy-preserving AI system capable of performing all detection and billing logic directly on edge hardware.\newline
				
			}
			
			\block{Challenges}
			{
			Although license plate detection systems have advanced rapidly, several persistent obstacles still hinder efficient deployment on low-power edge devices.\newline
			
			\begin{itemize}
				\item \textbf{Environmental and Lighting Variability}\\
				Real-world environments with variable outdoor conditions may lead to illumination differences. Sunlight, surface reflections, mist, and precipitation can significantly affect the accuracy of license plate recognition and the extraction of alphanumeric characters. \newline Studies have shown that varying brightness and weather conditions increase character recognition error rates. Airports operating throughout the day may experience low-visibility conditions, leading to inaccuracies in text extraction.\newline
				
				\item \textbf{Edge Device Compute Limitations}\\
				Edge devices such as the \textbf{NVIDIA Jetson Nano} are constrained in GPU computation power and memory availability.\newline High-performance deep-learning models demand significant computational resources, which can reduce frame rates below real-time levels. Prolonged GPU utilization may also cause overheating and performance throttling during continuous operation.\newline
				
				\item \textbf{False Positives and Non-Standard Plates}\\
				Incorrect detections occur when objects with plate-like textures (advertisements, reflections, or vehicle logos) are misidentified as license plates. These inconsistencies increase storage overhead and reduce system credibility in enforcement scenarios. \newline
				
				\item \textbf{Cost and Scalability}\\
				Commercial systems are often expensive, closed-source, and require recurring maintenance fees. Studies on edge-based license plate recognition deployments emphasize the need for modular, affordable architectures to enable intelligent surveillance.\newline
				Such compliance stress on the need for using minimal data, responsible usage under a legal purview, and transparent operation. \newline
				Systems processing images without anonymity create risks that are non-compliant for public infrastructure deployments.\newline
			\end{itemize}
			
				
			}
		}

			\block{Solutions}
			{
			
			The project aims to address key challenges in real-world license plate recognition by combining hardware-aware optimization, efficient AI system design, and strict adherence to legal and privacy requirements.
			
			\begin{itemize}
				\item \textbf{Addressing Environmental and Lighting Variability}
				\begin{itemize}
					\item Apply dynamic image preprocessing to normalize brightness, gamma, and contrast across frames.
					\item Integrate IR-assisted illumination and adaptive exposure for stable nighttime operation.
					\item Incorporate datasets with wide variation across daytime, nighttime, and fluctuating weather conditions to improve detection accuracy.
				\end{itemize}
				
				\item \textbf{Optimizing Computational Performance}
				\begin{itemize}
					\item Employ multi-threaded and asynchronous system pipelines to enable parallel execution of camera capture and optical character recognition.
				\end{itemize}
				
				\item \textbf{Reducing False Positives and Handling Non-Standard Plates}
				\begin{itemize}
					\item Introduce temporal validation across consecutive frames to confirm detection stability.
					\item Apply bounding-box correlation between vehicle and plate regions to prevent spurious matches.
					\item Include a classification module for special license plates (e.g., emergency or diplomatic) based on prefix and color cues.
				\end{itemize}
				
				\item \textbf{Ensuring Privacy and Legal Compliance}
				\begin{itemize}
					\item Hash recognized license plate numbers using \textbf{SHA-256} before database storage to ensure irreversible anonymization.
					\item Retain images temporarily ($\leq$ 24 hours) with automated deletion through a background cleanup scheduler.
					\item Encrypt all local \textbf{SQLite} databases and restrict external network communication.
				\end{itemize}
			\end{itemize}
			
			}
			
			\block{Results}
			{
			The implemented license plate detection system achieved stable real-time performance on the NVIDIA Jetson Nano. The optimized detection and character recognition pipeline consistently identified license plates across multiple vehicle types and plate formats, maintaining reliable inference speeds suitable for continuous edge-based operations. \newline
			
			Detection accuracy remained robust under varying environmental conditions, including changes in illumination, partial occlusions, and moderate motion blur. 
			
			The system generated actionable outputs through real-time visualization and structured logging of detected plates. Analysis of recognized license plates further supports downstream applications such as access control, parking management, and traffic monitoring. \newline
			
			An end-to-end automated workflow enabled seamless operation from video acquisition to license plate recognition and result display. The lightweight edge deployment minimized latency and power consumption while preserving data privacy, making the system suitable for long-duration, on-site deployment in resource-constrained environments.\newline \newline 
			}
